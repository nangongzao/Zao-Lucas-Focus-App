  1. 注册/登录 GitHub 👉 打开 https://github.com
    没账号就注册一下，很快的～

  2. 新建一个仓库

    • 名字随便（比如 focus-nest）

    • 记得勾上 Public，别勾初始化 readme（optional）

  3. 上传代码

    • 打开刚才创建的仓库

    • 点击 Add file → Upload files

    • 上传你这个 .html 文件（你可以保存它为比如 index.html）

  4. 启用 GitHub Pages

    • 点仓库里的 Settings → 找到左边栏 Pages

    • Source 选择 Deploy from a branch

    • Branch 选 main，然后 / (root)

    • 保存之后，下面就会出现一个链接，比如：
      https://你的用户名.github.io/focus-nest/
